#  Linear Search

##  What is Linear Search?

Linear Search is the simplest searching technique.

It checks each element **one by one**, from start to end, until:

- The target value is found  
- Or the list ends  

It does not require the array to be sorted.

---

##  Real-Life Example

Imagine you're searching for your roll number in a printed attendance sheet.

You:

- Start from the top  
- Check each name one by one  
- Stop when you find your name  

You don’t jump randomly — you go sequentially.

That’s exactly how **Linear Search** works.
